The Big Shift

从算法时代，到系统时代

如果把 AI 看成一条能力链，那么每一次范式跃迁，本质上都在回答一个更高阶的问题：机器能不能从数据里学？能不能自己提取表示？能不能用语言压缩世界？能不能像一个执行者一样调用工具？能不能在行动之前先构造一个内在世界？

Machine Learning

从规则到统计

机器不再只依赖人工写死的规则，而是开始从数据中拟合模式，形成第一代可规模化预测系统。

Deep Learning

从特征到表示

深度神经网络把“表示学习”变成核心，让系统能够端到端地从大规模数据中抽取层级特征。

LLM

从任务到通用语言接口

语言模型把自然语言变成统一控制面板，让“写提示词”逐步取代“为每个任务单独造界面”。

Agents & Beyond

从回答到行动

当模型开始接入工具、记忆与环境，它就不再只是聊天对象，而是任务执行系统的核心组件。

Timeline

AI 演进时间线

这条时间线聚焦的是“能力接口”如何变化：从特征工程，到表示学习，到语言接口，再到可观测、可治理的代理系统。

1950s–2000s

ML / Machine Learning

机器开始从样本中学习，而不只靠人手写规则

统计学习把 AI 从纯符号推理拉向 data-driven 方法。线性模型、决策树、SVM、贝叶斯方法与特征工程，构成了现代 AI 最早的工业基础。

  • 关键词：supervised learning、feature engineering、generalization
  • 代表意义：AI 第一次具备可规模化部署的预测能力
2012

DL / Deep Learning

深度网络让“表示学习”成为主角

AlexNet 及其后续浪潮让业界重新认识到深层神经网络的力量。视觉、语音、推荐系统开始全面受益于大模型、大数据与 GPU。

  • 关键词：CNN、backpropagation、GPU acceleration
  • 代表意义：机器不仅会拟合，还能自己学出更强特征
2017–2023

LLM / Large Language Models

语言成为通用接口，模型开始具备跨任务迁移能力

Transformer 改写了序列建模范式。随后，大规模预训练与指令对齐让模型从“语言预测器”进化为“通用文本操作系统”。

  • 关键词：Transformer、pretraining、instruction tuning
  • 代表意义：自然语言第一次变成大规模的人机协作接口
2024

Agent Era

AI 从“会说”走向“会做”

当模型开始调用工具、使用浏览器、读写文件、维护状态并分解任务，agent 时代就开始了。一个回答不再是终点，而是执行流程的入口。

  • 关键词：tool use、memory、workflow orchestration
  • 代表意义：模型成为行动链条中的推理核心
2025–2026

OpenClaw

开放个人 agent 形态，把 AI 拉近到“我的设备、我的通道、我的工作流”

OpenClaw 代表的是这一波 personal AI assistant 的一个清晰方向：agent 不只是网页上的聊天框，而是一个运行在用户自己环境中的常驻系统，能连接消息、文件、浏览器、语音与自动化动作。

  • 关键词：local control、multi-channel assistant、always-on agent
  • 代表意义：agent 从 demo 走向真实用户工作流
2026

Harness Engineering

模型外面的工程层，开始成为真正的竞争壁垒

Harness Engineering 是 agent 生产化语境中的新关键词。它强调：可靠 agent 不只是模型本身，而是包裹在外面的工具接口、上下文管理、状态机、监控、评估、安全与治理系统。

  • 关键词：observability、guardrails、evaluation、governance
  • 代表意义：AI 进入“系统工程”阶段，而不再只是“提示工程”阶段
Next

World Models

下一站，可能不是更会说，而是更会预演真实世界

如果未来的 AI 需要更稳定地规划、试错、长期行动，它就可能需要某种 world model： 一个能压缩环境、预测变化、在行动前进行内部模拟的世界表征层。

  • 关键词：simulation、latent planning、environment modeling
  • 代表意义：AI 或将从“语言推理”进一步走向“世界推理”

System Age

从模型到系统，差距是怎么被拉开的

进入 agent 时代后，真正决定体验与可靠性的，越来越不是底座模型参数量，而是它被怎样约束、观测、编排、测试与治理。

OpenClaw

开放、个人、常驻的 agent 形态

在这波 agent 浪潮里，OpenClaw 代表的是“用户主权型 agent”的想象：AI 运行在你的设备和你的渠道里，服务你的消息流、文件流和工作流，而不是只存在于一个封闭聊天窗口。

personal assistant own your workflow tool-connected

Harness Engineering

没有 harness，只有模型；有了 harness，才是产品

Harness 是把模型变成系统的中间层。它定义工具边界、管理上下文、记录状态、验证输出、处理失败，并在需要时强制人类确认。

01 Model

语言、视觉、多模态推理能力

02 Tools & Skills

浏览器、文件、邮件、代码、数据库、外部 API

03 Memory & Context

短期上下文、长期记忆、任务状态与角色约束

04 Control & Safety

权限、确认机制、审计日志、回退与防注入策略

05 Evaluation & Ops

可观测性、测试、指标、失败恢复与持续改进

Future Horizon

未来可能长什么样

如果把今天的 agent 看成“会调用工具的语言系统”，那么未来的 AI 很可能是“会理解环境、会预演后果、会长期协调资源的行动系统”。

World Models

更强的环境表征

AI 不只预测下一个 token，还尝试压缩“世界会怎样变化”，从而支持更稳的规划与试错。

Embodied Agents

从数字空间走向物理空间

当 agent 连接机器人、传感器和实时控制系统，它将面对更复杂的时间、空间与安全约束。

Multi-Agent Systems

从单智能体到组织级协作

未来系统可能不是一个超级助手，而是一组可通信、可分工、可审计的 agent 网络。

Governed Intelligence

治理会成为一等公民

权限、审计、可解释性、责任边界与人类确认，将和模型能力同样重要。

内容注记

本页将 OpenClaw 与 Harness Engineering 视为 2025-2026 年 agent 时代的代表性术语；World Models 则被视为下一阶段值得重点关注的方向，强调的是“模拟、预演、规划”能力，而不是单纯扩大语言模型规模。
