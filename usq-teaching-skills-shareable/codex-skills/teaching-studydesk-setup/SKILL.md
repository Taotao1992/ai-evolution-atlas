# Teaching StudyDesk Setup

## Overview

Set up StudyDesk in small, reviewable batches. Inspect the current shell first, apply only the requested configuration, and verify the student-facing result before moving to the next section.

## Workflow

1. Confirm the setup scope.
   Identify the course shell, teaching period, task owner, and whether this is an initial setup or a patch to an active site.

2. Review the reference guide.
   Read [references/panopto-guide.md](references/panopto-guide.md).
   If the reference video is inaccessible, use the workflow below and surface any missing policy or course-specific settings.

3. Inventory the current shell.
   Check topic sections, visibility, assignment activities, due dates, submission types, group mode, and any gradebook linkage that the requested change may affect.

4. Apply changes in narrow batches.
   Update one area at a time such as section visibility, a single assignment, or one block of dates.
   If the task is specifically about project group imports, prefer the separate `studydesk-group-import` skill instead of rebuilding that workflow here.

5. Verify from the learner perspective.
   Re-open the edited areas and confirm visibility, due dates, submission settings, and group behavior line up with the request.

6. Summarize the final state.
   Report what was changed, what was verified, and any remaining items that still need course-staff confirmation.

## Safety Checks

- Stop before deleting live activities or bulk-publishing large areas unless the user clearly wants that outcome.
- Do not guess due dates, penalties, or release windows; confirm them from the course instructions or the user.
- For active courses, mention any setting that could immediately affect students or markers before applying it.

## Execution Notes

- Use the Computer Use plugin for live StudyDesk actions.
- Prefer preparing a checklist first when the request is broad or when the course shell is already in active use.
- If exports or grade tables are needed during setup verification, use spreadsheet tooling instead of manual calculations.
