# StudyDesk Setup Reference

- Title: `Set up studydesk`
- URL: [Panopto Viewer](https://usq.au.panopto.com/Panopto/Pages/Viewer.aspx?id=98a53c84-8e6f-4ee8-b362-b0b200127aee)
- Access note: This guide appears to require UniSQ/Panopto login. If the page is not readable in the current environment, treat it as a source of UI terminology and use the checklist below.

## Core Setup Checklist

1. Confirm the correct course shell and offering.
2. Review hidden vs visible sections and activities.
3. Check each assignment's dates, submission type, and group mode.
4. Confirm gradebook linkage or grade item behavior if assignments are touched.
5. Verify the student-facing view after each major settings change.

## Clarify Before Making Changes

- Is this a fresh shell setup or a live-course correction?
- Which activities or sections are in scope?
- Are groups, groupings, or assignment restrictions involved?
- Are there approved dates or policies that must be copied exactly?
