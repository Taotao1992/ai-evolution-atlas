# Teaching Finalise Grades And Return

## Overview

Treat grade finalisation as high stakes. Reconcile the data first, verify anomalies before changing anything, and pause immediately before any irreversible submission or release action.

## Workflow

1. Confirm authority and scope.
   Identify the course, offering, grading scale, deadline, and whether the request is to prepare, verify, or actually submit the final return.

2. Review the reference guide.
   Read [references/panopto-guide.md](references/panopto-guide.md).
   If the guide is inaccessible, continue with the audit workflow below and do not invent institutional steps.

3. Reconcile marking completion.
   Check that all marking is complete, special cases are identified, withheld or exempt students are known, and hidden/manual grade items are understood.

4. Verify grade data.
   Export or inspect the grade table as needed.
   Look for missing grades, out-of-range values, inconsistent overrides, rounding issues, and moderation adjustments that were not applied cleanly.

5. Prepare the examiner return.
   Populate the required summary carefully, note unresolved exceptions, and keep a brief audit trail of counts, exports, and checks performed.

6. Pause before final submission.
   Immediately before submitting the return, posting final grades, or making a live change that affects results, reconfirm the exact target course and the user's intent.

7. Verify the final state.
   Re-open the relevant page or export after submission and confirm statuses, totals, and any generated confirmation markers.

## Safety Checks

- Never assume hurdle, rounding, or withheld-result policy.
- Do not release or submit if unresolved anomalies remain.
- Prefer a dry-run summary when the user has not explicitly asked for the final live action.

## Execution Notes

- Use spreadsheet tooling for grade exports or reconciliation instead of manual arithmetic.
- Use the Computer Use plugin for live StudyDesk or institutional web workflows.
- If another staff member's approval is required, stop at the ready-to-submit state and report exactly what remains.
