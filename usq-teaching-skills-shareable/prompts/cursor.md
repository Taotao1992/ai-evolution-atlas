Use the UniSQ Teaching Skills Pack in this workspace.

Start by reading:

- `README.md`
- `docs/teaching-workflows.md`
- `docs/video-sources.md`

If the task is about StudyDesk group import, also read:

- `docs/workflow.md`
- `docs/troubleshooting.md`

Next:

1. Pick the relevant workflow for the task.
2. For group-import work, run `scripts/build_group_import.py` and generate:
   - a clean `idnumber,groupname` CSV
   - an issues report
3. For teaching-operation work, follow the documented workflow instead of guessing the order of steps.
4. If browser tools are available, help me complete the live steps carefully.
5. If browser tools are not available, give me a precise manual checklist.
6. Stop before risky live actions unless I clearly ask for them, especially:
   - releasing results
   - approving examiner return
   - widening Panopto sharing
   - bulk-releasing course content
7. Treat grade data as confidential.

Prefer safe, auditable steps and call out anything that could affect live students in a non-obvious way.
